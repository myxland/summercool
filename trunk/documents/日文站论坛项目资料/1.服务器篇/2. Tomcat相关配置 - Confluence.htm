1.Tomcat服务器的安装

    A. 首先安装JDK，如果已经安装那么请跳转至第H步。

    B. 下载JDK5，并安装到本地目录，如：D:\jdk1.5.0_16。

       

    C. 配置JDK到环境变量。

    D. 打开系统的环境变量设置界面，点击新增按钮，输入变量名为JAVA_HOME，变量值为JDK的安装路径如：D:\jdk1.5.0_16，点击确定。

       

    E. 添加到PATH中，选择PATH，点击编辑，在变量值后添加%JAVA_HOME%/bin，注意和前面的值用;分隔。

       

    F. 打开命令提示符工具，输入java -version，回车，如果返回Java版本信息，说明JDK安装成功，否则请检查上面的过程。

       

    G. 安装出口限制安全包，安装在JDK安装目录下的\jre\lib\security目录下，如：D:\jdk1.5.0_16\jre\lib\security，覆盖原始文件。

       

    H. 下面开始安装Tomcat服务器。

    I. 下载Tomcat，并解压到本地目录，如：D:\apache-tomcat-6.0.20。

        

    J. 复制mail.jar, activation.jar, j2ee.jar放置到Tomcate目录下的lib目录中。

       

    K. 现在Tomcat服务器已经安装好了。

2.Tomcat服务器的启动

    A. 修改Tomcat服务器的启动是内存参数。

        1). 打开Tomcat的启动脚本文件catalina.bat或catalina.sh，在Tomcat安装目录的bin目录下。

            

        2). 在JAVA_OPTS的注释后添加set JAVA_OPTS=-Xms256m -Xmx256m，具体值根据需求调整。

rem    JAVA_OPTS    (Optional) Java runtime options used when the "start",
rem                 "stop", or "run" command is executed.
rem
set JAVA_OPTS=-Xms256m -Xmx256m

    B. 修改Tomcat服务器的启动ROOT目录

        1). 打开Tomcat的服务器配置文件server.xml，在Tomcat安装目录的bin目录下。

            

        2). 在<host>节点下添加以下代码，其中docBase的值为项目部署目录，path的值为应用上下文，这里设置为空，代表可以通过根路径直接访问。

<Context path="" docBase="D:\etc\shinkansen\deploy\target\shinkansen" debug="0" reloadable="true" />

    C. 现在运行Tomcat安装目录下的bin目录下的startup.bat或startup.sh文件就可以启动Tomcat服务器，访问应用程序了。

       

3.Tomcat服务器的配置

    A. 编码配置。

       说明：1. 打开Tomcat的服务器配置文件server.xml，在Tomcat安装目录的bin目录下。
                 2. 修改Connector节点，增加useBodyEncodingForURI="true"和URIEncoding="UTF-8"属性。

<Connector connectionTimeout="20000" port="8080" protocol="HTTP/1.1" redirectPort="8443" useBodyEncodingForURI="true" URIEncoding="UTF-8" />
